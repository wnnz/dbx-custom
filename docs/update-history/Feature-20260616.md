## 功能概览

### 1. SQL Server Windows 身份验证

- 新建或编辑 SQL Server 连接时，支持选择 Windows 身份验证。
- 选择 Windows 身份验证后，不再展示用户名和密码输入框。
- 提交连接配置、测试连接和读取可见数据库时，会清空用户名和密码，避免误用 SQL Server 登录凭据。

涉及文件：

- `apps/desktop/src/components/connection/ConnectionDialog.vue`

### 2. SQL 编辑器选中文本阴影修复

- 修复 CodeMirror 文本选择层在行尾零宽范围上错误渲染阴影的问题。
- 解决从下方向上选择 SQL 文本时，未实际选中的行尾文本也出现选中阴影的异常。
- 新增单元测试覆盖非空行行尾零宽范围、实际选中文本、空行选中等场景。

涉及文件：

- `apps/desktop/src/lib/codemirrorTrimmedSelectionLayer.ts`
- `apps/desktop/src/lib/__tests__/codemirrorTrimmedSelectionLayer.spec.ts`

### 3. 表和视图生成实体代码

- 在表和视图右键菜单中新增“生成实体代码”入口。
- 侧边栏树和对象浏览器两个入口都支持该功能。
- 新增实体代码弹框，支持选择语言和 ORM，并支持复制、保存生成结果。
- 生成代码会基于表/视图字段元数据输出字段类型、主键、nullable、字段映射和注释信息。
- 当前支持的语言与 ORM：
  - C#：EF Core、SqlSugar
  - Java：JPA / Hibernate、MyBatis Plus
  - TypeScript：TypeORM
  - Go：GORM
  - Python：SQLAlchemy
- 新增实体代码生成器单元测试，覆盖 EF Core、SqlSugar、TypeORM 的注释和字段映射输出。

涉及文件：

- `apps/desktop/src/lib/entityCodeGenerator.ts`
- `apps/desktop/src/components/objects/EntityCodeDialog.vue`
- `apps/desktop/src/components/sidebar/TreeItem.vue`
- `apps/desktop/src/components/objects/ObjectBrowser.vue`
